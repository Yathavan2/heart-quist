# heart-quist
Heart Quiz Game is an interactive and engaging web-based quiz application designed to test users' knowledge while providing a fun and visually appealing experience. The game features a series of questions with instant feedback, a scoring system, and a simple, user-friendly interface.


✨ Features
Interactive quiz gameplay with multiple-choice questions
Real-time score tracking
Instant feedback on answers
Simple and responsive user interface
Lightweight and easy to run on any browser


🛠️ Technologies Used
HTML
CSS
JavaScript


🚀 How to Run
Clone the repository
Open index.html in your browser
Start the quiz and enjoy!


🎯 Purpose
This project was developed to enhance front-end development skills and demonstrate interactive UI design and JavaScript logic implementation.
